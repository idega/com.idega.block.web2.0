Method Detail

isStrechable

boolean isStrechable()
    A Circle can't streched. In this case this method returns always false. So - no resize handles at the top, bottom,left and the right are visible.
    Returns:
      Returns always false. It is not possible to strech a circle. Use draw2d.Oval instead.

setDimension

void setDimension( <int> w, <int> h)
    It is not possible to set different values widht and height for a circle. The greater value of w and h will be used only.
    Parameters:
      w - The new width of the circle.
      h - The new height of the circle.
    See:
      - draw2d.VectorFigure.setDimension()
